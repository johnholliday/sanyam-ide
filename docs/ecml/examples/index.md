# Examples

Learn ECML through practical examples. These examples progress from basic concepts to advanced patterns.

## Basic Examples

Start here to understand the fundamental building blocks.

### [Basic Actor](/examples/basic-actor/)
Define a simple actor representing a role in your enterprise.

### [Basic Content](/examples/basic-content/)
Define a content item representing a document or artifact.

### [Basic Permission](/examples/basic-permission/)
Define a permission for access control.

## Intermediate Examples

Build on the basics with more complex structures.

### [Activity with Tasks](/examples/intermediate-activity-with-tasks/)
Model a business process with nested tasks and properties.

### [Security Model](/examples/intermediate-security-model/)
Configure security groups, permissions, and compliance labels.
