# ECML Language

The Enterprise Content Modeling Language provides a rich set of constructs for modeling enterprise content, workflows, and security.

## Documentation

- **[Language Reference](/language/reference/)** - Complete syntax and semantics
- **[Quick Reference](/language/quick-reference/)** - Syntax cheatsheet
- **[Tutorial](/language/tutorial/)** - Step-by-step learning guide

## Core Concepts

ECML is built around several core concepts:

### Actors
People, roles, or systems that participate in your enterprise processes.

### Content
Documents, files, and data artifacts that flow through your system.

### Activities & Tasks
Business processes and their component steps, organized hierarchically.

### Security & Compliance
Groups, permissions, and labels that control access and retention.

### Workflows
Orchestration of activities with conditional logic and repetition.
